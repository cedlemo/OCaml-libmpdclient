Module Mpd.LwtConnection

Offer functions and type in order to handle connections to the mpd server at the socket level in Lwt thread.

type t

Lwt connection type for thread usage

val initialize : string ‑> int ‑> t option Lwt.t

Create the connection in a Lwt thread, returns None if the connection can not be initialized.

val write : t ‑> string ‑> int Lwt.t

Write in a Mpd connection throught a Lwt thread. Return -1 if an it fails with an exception.
