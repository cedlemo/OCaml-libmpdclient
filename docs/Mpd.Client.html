Module Mpd.Client

module Client: sig .. end
Provides functions and type in order to communicate to the mpd server with commands and requests.
type c = {
   connection : Mpd.Connection.c;
   mpd_banner : string;
}
Client type
val initialize : Mpd.Connection.c -> c
Initialize the client with a connection.
val send : c -> string -> Protocol.response
Send to the mpd server a command or a request. The response of the server is returned under the form of a Protocol.response type.
val mpd_banner : c -> string
Return the mpd banner that the server send at the first connection of the client.
val status : c -> Status.s
Create a status request and returns the status under a Mpd.Status.s type.
val ping : c -> Protocol.response
Does nothing but return "OK".
val password : c -> string -> Protocol.response
This is used for authentication with the server. PASSWORD is simply the plaintext password.
val tagtypes : c -> string list
Shows a list of available tag types. It is an intersection of the metadata_to_use setting and this client's tag mask. About the tag mask: each client can decide to disable any number of tag types, which will be omitted from responses to this client. That is a good idea, because it makes responses smaller. The following tagtypes sub commands configure this list.
val close : c -> unit
Closes the connection to MPD. MPD will try to send the remaining output buffer before it actually closes the connection, but that cannot be guaranteed. This command will not generate a response.
